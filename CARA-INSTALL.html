📲 Cara Install ServisTek sebagai Aplikasi

PWA — Progressive Web App
ServisTek menggunakan teknologi PWA — bisa diinstall seperti APK di HP Android, dan seperti aplikasi di PC/laptop. Tidak perlu Play Store atau App Store!

📱 Install di HP Android (Chrome)

1. Buka file index.html di browser Chrome (atau hosting di server lokal)
2. Akan muncul tombol "📲 Install App" di pojok kanan atas — klik tombol itu
3. Atau: klik menu ⋮ (titik tiga) di Chrome → pilih "Tambahkan ke layar utama" atau "Install aplikasi"
4. Konfirmasi → ServisTek akan muncul di homescreen seperti APK asli ✅

💻 Install di PC / Laptop (Chrome atau Edge)

1. Buka index.html di browser Chrome atau Microsoft Edge
2. Di pojok kanan address bar akan muncul ikon ➕ install — klik
3. Atau: menu Chrome → "Cast, Save, and Share" → "Install ServisTek"
4. Aplikasi akan terbuka di jendela tersendiri seperti software biasa ✅

🔄 Cara Kerja Auto-Update

Otomatis: ServisTek mengecek update setiap 30 menit saat aplikasi terbuka
Banner: Jika ada versi baru, muncul banner hijau di bagian bawah: "Update ServisTek tersedia!"
Update: Klik "Update Sekarang" → halaman refresh otomatis dengan versi terbaru
Data aman: Semua data riwayat servis, stok, dan pelanggan tersimpan di localStorage — tidak hilang saat update
💡 Untuk update manual: refresh halaman dengan Ctrl+Shift+R (hard refresh) di PC, atau close dan buka kembali di HP.

🌐 Cara Buka di HP Lain / Share ke Teman

Opsi 1 — File lokal: Copy folder servistek-pwa ke HP/PC lain, buka index.html dengan Chrome
Opsi 2 — Server lokal: Jalankan di server lokal (XAMPP, Nginx, dll) lalu akses via IP jaringan WiFi yang sama
Opsi 3 — Hosting: Upload ke hosting (Netlify, GitHub Pages, dll) — gratis — lalu bagikan link-nya
⚠️ Service Worker (auto-update) HANYA berjalan jika dibuka melalui server (http:// atau https://), bukan langsung dari file (file://). Untuk file lokal tanpa server, fitur update tidak aktif tapi aplikasi tetap bisa dipakai normal.

📦 Isi Folder ServisTek PWA

index.html — Aplikasi utama ServisTek v11
manifest.json — Konfigurasi PWA (nama, ikon, tema)
sw.js — Service Worker (offline + auto-update)
icons/ — Ikon aplikasi berbagai ukuran (72px — 512px)
CARA-INSTALL.html — Panduan ini
